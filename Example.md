The animation component can takes these props:
* **duration**
```javascript

<Entrance duration="2s">

```
* **delay**
```javascript

<Entrance delay="2s">

```
* **timingFunction**
```javascript

<Entrance timingFunction="ease-in">

```
* **direction**
```javascript

<Entrance direction="reverse">

```
* **iterations**
```javascript

<Entrance iterations="2">

```
* **backfaceVisible**
```javascript

<Entrance backfaceVisible="visible">

```
* **fillMode**
```javascript

<Entrance fillMode="forward">

```
* **playState**
```javascript

<Entrance playState="running">

```
* **block**
```javascript

<Entrance block>
  <h1>
    Hello World
  </h1>
</Entrance>

```
